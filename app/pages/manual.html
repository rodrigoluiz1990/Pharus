Manual do Sistema

Menus Laterais

Clique em um menu para ver a explicação da tela.

Dashboard

Resumo rápido do que está acontecendo agora no sistema.

Para que serve

Mostrar os números principais: quantidade de tarefas, atrasos, entregas próximas e concluídas.

O que você encontra

  • Cards com totais.
  • Gráficos de status, prioridade e tipo.
  • Listas de top responsáveis e top clientes.
  • Bloco de próximas entregas.

Passo a passo

  1. Abra o Dashboard ao entrar no sistema.
  2. Verifique primeiro o card de atrasadas.
  3. Veja os gráficos para entender onde está o maior volume.
  4. Use a lista de próximas entregas para planejar o dia.

Agenda

Organização de compromissos por dia, semana e mês.

Para que serve

Controlar eventos e tarefas com data, evitando esquecimentos e atrasos.

Como usar

  1. Clique em Novo evento.
  2. Preencha título, tipo e status.
  3. Defina data e horário ou marque Dia inteiro.
  4. Salve e confira no calendário.

Dica prática

Use a visão semanal para planejar o curto prazo e a mensal para acompanhar feriados e picos de demanda.

Quadro de Avisos

Comunicação interna com mensagens importantes para a equipe.

Quando usar

Use para recados oficiais, alertas de processo, janelas de manutenção e comunicados gerais.

Publicando um aviso

  1. Clique em Novo aviso.
  2. Informe título, prioridade e conteúdo.
  3. Escolha grupo específico ou todos os grupos.
  4. Defina data limite de visibilidade e salve.

Boa prática

Escreva textos curtos, com ação objetiva: o que mudou, quem precisa agir e até quando.

Quadro de Tarefas

Tela principal de operação para criar, acompanhar e concluir tarefas.

Visões disponíveis

  • Quadro: cards por status.
  • Tabela: comparação rápida em linhas e colunas.
  • Título: foco em lista simples.

Fluxo básico

  1. Clique em Nova tarefa.
  2. Preencha título, tipo, prioridade, responsável e prazo.
  3. Use filtros para localizar tarefas específicas.
  4. Atualize o status conforme o andamento.

Importação em lote

Ideal para carga inicial: baixe o modelo, preencha, revise a pre-visualização e confirme a importação.

Clientes

Cadastro completo de empresas atendidas e dados de contato.

Abas da tela

  • Geral: nome, sigla, contato e dados fiscais.
  • Endereço: CEP com busca automática.
  • Conexão remota: múltiplos acessos (AnyDesk e outros).

Passo a passo

  1. Abra Novo cliente.
  2. Complete os campos obrigatórios na aba Geral.
  3. Use o botão de buscar CEP para preencher endereço.
  4. Salve e valide se o cliente apareceu na lista.

Dica de qualidade

Padronize siglas (ex: BP, MVS, RV) para melhorar filtros e relatórios.

Atendimento

Registro de atendimento ao cliente com historico colaborativo.

Para que serve

Centralizar cada atendimento com titulo, descricao, status, prioridade, responsaveis, tarefas relacionadas e anexos.

Fluxo recomendado

  1. Clique em Novo atendimento.
  2. Informe titulo e contexto do atendimento.
  3. Selecione cliente e vincule um ou mais responsaveis.
  4. Relacione tarefas ja existentes e anexe evidencias.
  5. Use comentarios para registrar andamento entre operadores.

Boas praticas

  • Atualize o status sempre que houver mudanca relevante.
  • Mantenha comentarios curtos com data e proximo passo.
  • Vincule tarefas para manter rastreabilidade do atendimento.

Chat

Comunicação rápida entre operadores sem sair do sistema.

Funções principais

  • Lista de contatos com ordenação por mensagem recente.
  • Indicadores de mensagens não lidas.
  • Abertura por tela principal e modo bolha/post-it.

Como conversar

  1. Abra o menu Chat.
  2. Clique no contato desejado.
  3. Digite a mensagem e envie.
  4. Acompanhe notificações no menu lateral.

Boa prática

Use o chat para dúvidas curtas. Para decisões oficiais, registre no Quadro de Avisos.

Relatórios

Construa relatórios personalizados e compartilhe com outros usuários.

Aba Relatórios

Mostra os relatórios cadastrados. Clique em uma linha para abrir e visualizar os dados.

Aba Construtor

  1. Clique em Novo relatório.
  2. Defina nome, disponibilidade e filtros.
  3. Escolha colunas e ordenação.
  4. Valide o preview e salve.

Exportação

Use os botões da lista para gerar PDF ou Excel e compartilhar com a equipe.

Configurações

Área administrativa para ajustar regras, perfil da empresa e comportamento do sistema.

Guia rápida das abas

  • Geral: nome do projeto, menu personalizado e cores/ícones.
  • Permissões: grupos e matriz de acesso por tela.
  • Usuários: cadastro de operadores.
  • Extensão: instalação e pacote do modo post-it.
  • Tabela: ordem e largura de colunas.

Fluxo recomendado para implantação

  1. Cadastre grupos de permissão.
  2. Cadastre usuários e vincule cada grupo.
  3. Ajuste tipos/prioridades (cor e ícone).
  4. Configure a extensão dos operadores.

Importante

Se algo ficar inacessível, valide primeiro as permissões do grupo do usuário logado.

Atualização e Backup

Como verificar versão, criar backup e iniciar atualização do sistema.

Onde fica

Entre em Configurações e abra a aba Atualização e Backup.

Verificar atualização

  1. Clique em Verificar atualizações.
  2. Compare Versão atual e Última versão.
  3. Se houver nova versão, siga para atualização.

Gerar backup

  1. Clique em Gerar backup agora.
  2. Aguarde a confirmação no status e no log.
  3. Guarde o caminho do arquivo de backup informado na tela.

Atualizar sistema

  1. Clique em Atualizar sistema.
  2. Confirme a execução quando o sistema pedir.
  3. Acompanhe o andamento no bloco de log.
  4. Ao finalizar, atualize o navegador (Ctrl+F5).

Observação importante

Somente usuários com permissão de manutenção podem usar essa aba.
